# slot10_50_100

slot simulator demo：

遊戲數據模擬器，輸出各項統計資料。

## 環境需求
 Go 1.22+

## Build
go build -o slot10_50_100.exe .

## Run (example)
.\slot10_50_100.exe 或 go run .\slot10_50_100.go
